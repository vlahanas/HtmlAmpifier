# An HTML to AMP markup converter for PHP

Usage:

// create a new instance of ampifier
$ampifier = new ContentAmpifier();

// grab your html content from any source
$html = "<html>...</html>";

// convert it to amp
$amp = $ampifier->ampify($html);
